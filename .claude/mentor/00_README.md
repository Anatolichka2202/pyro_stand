# Менторский режим — pyro_stand

Эта папка — персональный учебный трек по проекту. Каждая сессия добавляет:
- разбор допущенных/найденных ошибок
- оценку хода мышления
- конкретные навыки к отработке

## Содержимое

| Файл | Что внутри |
|------|-----------|
| `session_01.md` | Разбор июньской сессии (задержка T0, баги таблицы, UX) |
| `learning_path.md` | Структурированный план обучения |
| `concepts.md` | Ключевые концепции проекта с объяснениями |
| `production.md` | Оценка готовности к продакшну |

## Как использовать

Перед новой сессией скажи Claude: "открой менторский режим".
Контекст из этой папки будет загружен и продолжит трек.
