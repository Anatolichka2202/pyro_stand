# Дизайн-мастерская / Design Collaboration Workspace

Эта ветка — канал связи между **Claude Design AI** (дизайнер) и **Claude Code** (оркестратор).  
Пользователь (@Anatolichka2202) просматривает и утверждает макеты.

---

## Рабочий процесс

```
Design AI → создаёт HTML-макет → коммитит в design/mockups
    ↓
Пользователь → смотрит, пишет комментарий "approved" / правки
    ↓
Claude Code → читает утверждённый макет → реализует в Qt6/C++ → открывает PR в master
    ↓
Design AI → проверяет реализацию через PR diff / скриншоты
```

## Текущее состояние приложения

**pyro_stand** — Qt6/C++20 desktop-приложение для контроля пуска ракеты.  
Язык интерфейса: **русский**.

### Текущий UI (что уже сделано)
- **Левая панель**: кнопки «Загрузить», «Запустить», «Стоп», «Сброс» + QTimeEdit для времени старта
- **Центр**: таблица событий (7 колонок: #, СОБЫТИЕ, КАНАЛЫ, ПЛАН МС, ФАКТ МС, ОТКЛ МС, СТАТУС)
- **Верхняя панель**: статус COM-порта `● COM7` и БЦВМ `● БЦВМ` (зелёный/красный)
- **Под таймером**: 8 точек-индикаторов каналов CH1–CH8 (серый → жёлтый → зел/красный)
- **Таймер**: большой, жёлтый/зелёный/красный в зависимости от фазы
- **Подсветка строки**: следующее ожидаемое событие выделяется тёмно-синим `#1c2d3a`

## Как добавить макет

1. Создать файл: `design/mockups/T{номер}-{название}.html`
2. В начале файла добавить статус:
   ```html
   <!-- STATUS: draft | approved | implemented -->
   <!-- TASK: T18 — Timeline widget -->
   <!-- NOTES: ... -->
   ```
3. Закоммитить в эту ветку и написать в комментарии PR что готово

### Требования к HTML-макетам
- Самодостаточный файл (всё в одном HTML, inline CSS+JS)
- Использовать цвета из таблицы выше
- Показывать **интерактивное** поведение (анимации, hover-эффекты) где возможно
- Размер окна: 1280×800 px (типичное окно приложения)
- Добавить текстовый блок внизу: «Qt-реализация: [краткое описание виджета и классов]»

---

## Статусы макетов
- `draft` — в работе, ещё не готов к ревью
- `approved` — пользователь одобрил, Claude Code берёт в реализацию  
- `implemented` — реализовано в Qt, PR смёрджен
