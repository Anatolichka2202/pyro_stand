## [3] Timeline (TimelineWidget, высота СТРОГО 52px)

```
─────────────────────────────────────────────────────────
    1    2   3  4                5     6
    │    │   │  │                │     │       ← вертикальные линии (1-2px)
────┼────┼───┼──┼────────────────┼─────┼──────  ← ось (1px #8b949e)
   -8с  -8с  0  0              +20с  +21с
                ↑ T0 (линия красная, чуть выше)
         ↑ синий playhead (1px, весь виджет по высоте)
```

**Текущие характеристики:**
- Высота: **52px** (fixed)
- Тики: **вертикальные линии** (1-2px), не кружки
- T0: красная линия #f85149, на 4px выше обычных тиков
- Playhead: синяя линия #58a6ff, 1px, **без свечения**
- Показывает только события с `hasChannels == true`
- Автомасштаб: rangeMin = min(events) − 2с, rangeMax = max(events) + 2с

⚠️ **Проблема:** тики слипаются при кластере (−8000, −7980, −7950, −7930 мс → 4 линии в 70мс окне, почти совпадают на экране)  
⚠️ **Проблема:** нет подписи "T0" над красной линией  
⚠️ **Проблема:** playhead без эффекта  
⚠️ **Проблема:** высота 52px — мало места для подписей

## Что нужно от Ланы (задание)

**T18 — Timeline** (приоритет 1):
Нарисуй редизайн виджета. Он фиксирован по высоте. Предлагаю 64px.
- Замени линии-тики на **кружки** (8–10px diameter)
- T0: красная вертикальная линия на всю высоту + подпись "T0" сверху
- Playhead: синяя линия + **glow** (rgba(88,166,255,0.4) вокруг)
- pending кружок: контур #8b949e, пустой внутри
- ok кружок: залитый #3fb950
- fail кружок: залитый #f85149
- Подписи: номер события над кружком, время под осью

**Панель управления** (приоритет 2):
- Разбей кнопки на визуальные группы внутри одного блока
- Пример группировки:
  ```
  ┌─ ПОДГОТОВКА ───────────────┐
  │ [▤ ЗАГРУЗИТЬ НА БОРТ]      │
  │ [⏱ УСТАНОВИТЬ ВРЕМЯ]       │
  │   00:00:10                 │
  ├─ УПРАВЛЕНИЕ ───────────────┤
  │ [■ СТОП]    [↺ СБРОС]     │
  └────────────────────────────┘
  ● COM7  ● БЦВМ  ← крупнее, вверху
  ```

**Лог** (приоритет 3):
- 2 колонки: `[10:00:22]` | `[сообщение]`
- Левая цветная полоска по типу: синяя=info, зелёная=ok, красная=fail
