# UI Specification — pyro_stand

Референс для разработки и будущих изменений UI.

## Цветовая палитра

| Роль | Hex | Применение |
|------|-----|-----------|
| Фон окна | `#0d1117` | `QMainWindow`, `QWidget` background |
| Поверхность панелей | `#161b22` | `QFrame`, карточки, лог |
| Бордер | `#30363d` | все рамки виджетов |
| Текст основной | `#e6edf3` | заголовки, значения |
| Текст вторичный | `#8b949e` | подписи, неактивное, INFO в логе |
| Жёлтый | `#e3b341` | обратный отсчёт, предупреждения |
| Зелёный | `#3fb950` | ОК, активное соединение, полёт |
| Красный | `#f85149` | FAIL, ошибка, СТОП |
| Синий акцент | `#58a6ff` | текущая строка таблицы (кол. 0), hover |
| Подсветка строки | `#1c2d3a` | pending-строка в таблице |

## Типографика

| Элемент | Шрифт | Размер | Weight |
|---------|-------|--------|--------|
| Таймер | JetBrains Mono | 52px | 600 |
| Заголовок фазы | системный | 11px | 600, uppercase |
| Кнопки | системный | 12px | normal |
| Таблица | системный | 11px | normal |
| Лог | JetBrains Mono (или системный mono) | 11px | normal |

## Кнопки

```css
/* Базовый стиль */
background: #0d1117;
border: 1px solid #30363d;
border-radius: 4px;
color: #e6edf3;
padding: 7px 12px;
font-size: 12px;
text-align: left;

/* Hover */
border-color: #58a6ff;
background: #1c2330;

/* СТОП (деструктивное действие) */
color: #f85149;
```

Кнопки с иконкой: `▤ ЗАГРУЗИТЬ НА БОРТ` / `⏱ УСТАНОВИТЬ ВРЕМЯ СТАРТА` / `■ СТОП` / `↺ СБРОС`

## Компоненты

### Таймер

```
мм:сс       — до 1 часа
чч:мм:сс    — 1 час и более
мм:сс.д     — в полёте (с десятыми)
СТОП        — после остановки
```

### Таблица событий

Колонки (7): `# | СОБЫТИЕ | КАНАЛЫ | ПЛАН МС | ФАКТ | ОТКЛ МС | СТАТУС`

Заголовок ФАКТ: `ФАКТ (ТИК)` во время полёта, `ФАКТ МС` после анализа.

Цвет отклонения (ОТКЛ МС):
- `≤ 5 мс` → `#3fb950`
- `6–50 мс` → `#e3b341`
- `> 50 мс` → `#f85149`

Статусы: `ожид.` (серый) / `OK ✓` (зелёный) / `FAIL ✗` (красный)

### Лог-панель

Формат строки:
```
▌  HH:mm:ss  УРОВЕНЬ  сообщение
```

Типы и цвета:
| Тип | Цвет полосы ▌ | Уровень | Цвет текста |
|-----|--------------|---------|-------------|
| system | `#30363d` | `INFO ` | `#8b949e` |
| event | `#3fb950` | `СОБЫТ.` | `#c8d0dc` |
| event-post | `#f85149` | `СОБЫТ.` | `#c8d0dc` |
| error | `#f85149` | `ОШИБКА` | `#f85149` |

### Индикаторы статуса (COM / БЦВМ)

Чип: `QFrame` с горизонтальным layout.
- Точка 9×9px: `#3fb950` (активно) / `#484f58` (нет связи)
- Подпись: название + строка статуса 9px `#8b949e`

### Таймлайн

- Высота: 72px
- Ось: 55% от верха
- Обычное событие: круг R=5
- T0 (LIFT_OFF_CONTACT): круг R=7, отдельный цвет
- Кластер (< 14px между событиями): один узел с числом
- Playhead: вертикальная линия с 3-слойным glow
