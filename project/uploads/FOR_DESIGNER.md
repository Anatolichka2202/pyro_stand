## Что за приложение

**pyro_stand** — десктоп-приложение (Windows, 1280×800) для контроля пуска ракеты.  
Оператор загружает циклограмму, запускает отсчёт, следит за срабатыванием пиропатронов.

## Задачи для дизайна (нужны HTML-макеты)

### ✅ ПРИОРИТЕТ 1 — T18: Таймлайн (уже в коде, нужен редизайн)
Текущий таймлайн простой. Хочется красивее:
- Горизонтальная полоса 52–64px
- События как тики с номерами
- T0 (старт) — красная линия выше остальных
- Синий playhead движется во время полёта
- Под тиками — время в секундах (`-8с`, `0`, `+20с`)

### 📋 ПРИОРИТЕТ 2 — Общий редизайн панели управления
Левая панель с кнопками выглядит скучно. Идеи:
- Сгруппировать кнопки визуально
- Статус COM/БЦВМ сделать более заметным
- Время старта — более акцентированно

### 📋 ПРИОРИТЕТ 3 — Лог-панель
Текущий лог — простой `QTextEdit`. Хочется:
- Чёткое разделение уровней (INFO / EVENT OK / EVENT FAIL / ERROR)
- Временны́е метки слева в отдельной колонке
- Горизонтальный разделитель между секциями

## Как создать макет

1. Создай файл `design/mockups/T{номер}-{название}.html`
2. Первые строки файла:
```html
<!-- STATUS: draft -->
<!-- TASK: T18 — Timeline redesign -->
```
3. Размер окна в макете: 1280×800px
4. Используй только цвета из таблицы выше
5. Добавь внизу комментарий: как это реализуется в Qt (виджет, класс, метод)

**Не читай другие файлы репозитория — всё что нужно есть в этом файле.**
